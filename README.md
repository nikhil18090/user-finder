# 🔍 Search Users

A dynamic user search application built using HTML, CSS, Tailwind CSS, and Vanilla JavaScript.

## 📌 Overview

Search Users is a frontend project that dynamically displays user profile cards and filters them in real time based on user input. The search results update instantly without refreshing the page.

## ✨ Features

* Dynamic user card generation
* Real-time search filtering
* Instant search results
* Animated "User Not Found" message
* Modern card-based UI
* Responsive layout
* Built with Vanilla JavaScript

## 🛠️ Technologies Used

* HTML5
* CSS3
* Tailwind CSS
* Vanilla JavaScript

## 📚 What I Learned
